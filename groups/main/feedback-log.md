# MPS Agent — Feedback Log

Log anonymised correction patterns here after each MPS session.
Format: `- agency: X | wrong: Y | correct: Z`

**Security rule:** No NRIC, no constituent names, no addresses,
no case reference numbers traceable to any individual.
Only policy corrections and case pattern observations.

---

## How to log feedback

During or after an MPS session, type in this chat:

```
/feedback [wrong thing agent said] → [correct answer] | agency: [HDB/CPF/MOH/MSF/MOM/ICA]
```

**Examples:**
```
/feedback EHG ceiling cited as $9,000 → correct ceiling is $8,000 for families | agency: HDB
/feedback MRSS match cap cited as $1,000 → correct cap is $600/year | agency: CPF
/feedback CHAS Blue per-capita threshold $700 → correct is $650/month | agency: MOH
/feedback stated fixed income threshold for PR → ICA has no fixed threshold, holistic assessment | agency: ICA
```

The agent will log each correction below in anonymised form.
This file is exported weekly to Hermes GEPA for skill improvement.
